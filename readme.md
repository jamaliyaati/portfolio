# Ebrahim Jamali — Portfolio

A personal portfolio website for Ebrahim Jamali, built as a static site with a dark editorial aesthetic inspired by avant-garde design portfolios.

## Key Technologies

- Pure HTML5, CSS3, and vanilla JavaScript — no build tools or frameworks required
- Google Fonts: Bebas Neue (display), Fraunces (serif), DM Mono (monospace)
- CSS custom properties for theming
- IntersectionObserver API for scroll-triggered animations

## File Structure

```
index.html   — markup and page structure
styles.css   — all styling and responsive layout
script.js    — interactivity: preloader, cursor, timeline, project filters
```

## Running Locally

Open `index.html` directly in a browser, or serve with any static file server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

No build step, no dependencies to install.
